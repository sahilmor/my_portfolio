**üèÅ Getting Started**
To get a local copy up and running, follow these simple steps.

Prerequisites

Make sure you have Node.js installed on your machine (v18 or higher is recommended for Astro). This project uses npm for package management.

Installation

Clone the repo:

Bash
git clone https://github.com/sahilmor/my_portfolio.git
Install NPM packages:

Bash
npm install
Running the Project

All commands are available in the package.json file:

Run a local development server:

Bash
npm run dev
Open http://localhost:4321 to view it in your browser. The page will auto-update as you make changes.

Build the site for production:

Bash
npm run build
This bundles the site into a dist/ directory.

Preview the production build locally:

Bash
npm run preview
This runs a local server to preview your static production site.
